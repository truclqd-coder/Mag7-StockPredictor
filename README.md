# 🚀 Mag7-Predictor: Stock Analysis & Forecasting

An end-to-end data science application that tracks, analyzes, and predicts the future returns of the **"Magnificent 7"** (AAPL, MSFT, GOOGL, AMZN, NVDA, META, TSLA).

## 📊 Live Dashboard
*This project uses Streamlit to visualize stock trends and LSTM model predictions.*

## 🛠️ Tech Stack
- **Language:** Python 3.9+
- **APIs:** Yahoo Finance (yfinance)
- **ML/Math:** TensorFlow (LSTM), Scikit-Learn, Pandas, NumPy
- **Visuals:** Plotly Interactive Charts
- **Framework:** Streamlit

## 📁 Project Structure
- `src/app.py`: The main dashboard application.
- `notebooks/`: Exploratory Data Analysis and model training scripts.
- `requirements.txt`: Python dependencies.
